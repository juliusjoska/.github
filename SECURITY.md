# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any of my projects, please report it responsibly.

**Email:** juliusjoska@protonmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

Only the latest version of each project is supported with security updates.
